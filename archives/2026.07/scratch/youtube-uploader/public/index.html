Google Cloud OAuth Guide

  1. Go to the Google Cloud Console.
  2. Create a new project and enable the YouTube Data API v3.
  3. Configure the OAuth consent screen and create OAuth 2.0 Client IDs (Application Type: Web Application).
  4. Set the Redirect URI to: http://localhost:3005/api/oauth2callback
  5. Add Client ID and Client Secret to the .env file in your project directory.
  6. Restart the server and click Connect YouTube Account.
